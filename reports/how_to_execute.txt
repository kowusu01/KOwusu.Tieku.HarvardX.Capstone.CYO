Instructions

Note: 
- This project was developded and tested in Windows. It has not been fully tested under other operating systems.
- RStudio is not required to execute the script.

To execute the script without RStudio:
- create a folder and name it e.g. base
- create a subfolder R under the base folder and copy the main script (adults_income.R) to that folder
- 
- open a command window using Administrator rights (right-click cmd icon and select Run As Administrator)
- change directroty to the base folder
- issue the following command: 
  rscript R/adults_income.R
